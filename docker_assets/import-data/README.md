# MySQL import data.

`dump.sql` is generated by Adminer.

It does the following.

- Creates the database.
- Drop all existing tables.
- Creates new tables, all four in total.
- Inserts all live data.

#### Adminer export instructions.

Visit: https://defacto2.net/data-inno?server=db&username=root

Select _defacto2-inno_.

Select _Export_ using the following options.

- Output: `save`
- Format: `sql`
- Database: `CREATE`
- Tables: `DROP+CREATE`
- Data: `INSERT`
