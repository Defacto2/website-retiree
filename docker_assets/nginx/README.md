# docker_assets nginx configurations

### `nginx/`

- `aliases.conf` Aliases to assets located on Docker shared volumes.

- `index.conf` Index for the Lucee/Tomcat configurations while testing.

- `legacy.conf` Legacy URL redirections.

- `locations.conf` Individual directory location configurations.

- `lucee_admin.conf` Block Lucee web accessible features.

- `nginx.conf` Root nginx configuration and MIME types.

- `rewrite.conf` Proxy rules and headers for testing.
